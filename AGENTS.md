# AGENTS.md

## 开发规则

- 将本仓库视为 Python 3.10+ 的标准库项目；不要新增第三方 Python 依赖，除非任务明确要求。
- 优先采用最小改动，不要做与当前任务无关的重构、拆文件或结构改写。

## 验证要求

- 修改 Python 代码后运行 `python3 -m pytest`；修改 CLI 参数、帮助文本、启动流程或运行时命令行为后，额外运行 `python3 task_pilot.py --help`。

## 兼容性与文档

- 保持现有的“单进程监督单个被监控会话”模型，以及 CLI 参数和配置字段语义的向后兼容，除非任务明确要求变更。
- 当改动影响用户可见的流程、命令或行为时，同步更新 `README.md` 和 `README.zh-CN.md`。
- 保持 `config.example.json` 与真实支持的配置一致；除非任务明确涉及配置变更，否则不要改变其字段含义。
